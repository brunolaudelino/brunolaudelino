Bordas

Podemos definir as configurações das nossas bordas através de propriedades

  • border-width
  • border-style
  • border-color

Exemplo:

Exemplo 1: border-width; 10px / border-style: solid
Exemplo 2: border-width:thick
Exemplo 3: double
Exemplo 4: border-style:groove
Exemplo 5: border-style:dotted, dashed
Exemplo 6: border-radius: 20px border-style:double
Exemplo 7: border-radius:50%, border-style:double
Exemplo 8: border-image-source: linear-gradient(red, orange, yellow);
